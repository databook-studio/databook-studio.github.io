Releases

Release history and feature updates for DataBook Studio.

[0.2.0] — 2026-07-11

Summary

Major update with SAYA AI copilot, Snowflake support, and workspace explorer.

Features

  • SQL Editor with semantic autocomplete
  • Interactive spreadsheet-style result grid
  • File system persistence for notebooks
  • Markdown support with rich formatting
  • PostgreSQL, MySQL, and Snowflake support

[0.1.0] — 2026-02-21

Summary

Initial release of DataBook Studio.

Features

  • SQL Editor with semantic autocomplete
  • Interactive spreadsheet-style result grid
  • File system persistence for notebooks
  • Markdown support with rich formatting
  • PostgreSQL and Snowflake support

Auto-Updates

DataBook Studio includes a built-in updater that checks for new versions and verifies each update before applying — so you always run a trusted build.

Local-first SQL workspace with a built-in AI copilot.
